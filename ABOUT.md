# About

## English

**Stock Analysis AI Platform** is a Streamlit-based project designed to help users explore stock market behavior through technical charting, signal analysis, next-day forecast experiments, and historical backtesting. The platform brings together market data visualization, trading signal logic, benchmark comparisons, confusion matrix evaluation, and class-level precision / recall / F1 metrics in one workspace.

It is built as a practical research and learning tool for users who want to test ideas, inspect price action, and evaluate how simple forecasting logic performs against historical data. The project is intended for educational and analytical use only and does not constitute financial advice.

## 中文

**Stock Analysis AI Platform** 是一個以 Streamlit 建立的股票分析專案，旨在幫助用戶透過技術圖表、訊號分析、下一交易日預測實驗及歷史回測，更深入了解股票市場行為。平台把市場數據視覺化、交易訊號邏輯、基準比較、confusion matrix 評估，以及每個類別的 precision / recall / F1 指標整合於同一個工作空間中。

此專案主要作為研究、學習及策略驗證工具，適合希望測試想法、觀察價格走勢，以及評估簡單預測邏輯在歷史數據中表現的用戶。此平台僅供教育與分析用途，並不構成任何投資建議。
